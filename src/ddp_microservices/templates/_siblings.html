{# Cross-service topnav box. Shows EVERY known sibling (the whole registry, pre-sorted by prefix in render_context). Link states for a non-self sibling: * a page CONTEXT view is present (charter/fond/archive) AND the sibling advertises it -> an enabled DEEP hand-off link (`///`); * NO context view (root / list page) -> an enabled HOME link (`//`); * a context view is present but the sibling does NOT advertise it -> GREYED, no link (its hand-off href is N/A here) -- but icon shown, since it is up; * DOWN -> GREYED, no link, prefix badge (its icon route is unreachable). Icons come from each peer's own `//icon.ico`. Origin (`absolute_links`, from proxy_url): false -> root-relative `//...` (single origin behind a front proxy); true -> absolute to each sibling's `base_url` (proxyless). Context: `handoff_type`/`handoff_value`, or `sibling_md5`. #} {%- set _ht = (handoff_type if (handoff_type is defined and handoff_type) else ('charter' if (sibling_md5 is defined and sibling_md5) else none)) -%} {%- set _hv = (handoff_value if (handoff_value is defined and handoff_value) else (sibling_md5 if (sibling_md5 is defined and sibling_md5) else none)) -%} {% for s in siblings %} {%- set _base = s.url if (absolute_links | default(false)) else '' -%} {%- set _has_ctx = _ht and _hv -%} {%- set _can = s.alive and (not s.is_self) and _has_ctx and (_ht in s.views) -%} {# enabled only if it can deep-link (context it supports) OR there is no context (-> home) #} {%- set _linkable = s.alive and (not s.is_self) and (_can or not _has_ctx) -%} {% if s.is_self -%} {# self: full-colour, non-clickable; its icon from its OWN canonical route #} {%- elif _linkable -%} {%- set _href = (_base ~ '/' ~ s.prefix ~ '/' ~ _ht ~ '/' ~ _hv) if _can else (_base ~ '/' ~ s.prefix ~ '/') -%} {%- elif s.alive -%} {# alive but does NOT support the current context view -> greyed, no link (href N/A); icon shown #} {%- else -%} {# down: its icon route is unreachable, so show a greyed prefix badge instead of a broken image #} {{ s.prefix }} {%- endif %} {% endfor %}
